# Grasshopper Project

`YingZao.GH` is the compiled Grasshopper plugin layer for the `yingzao` Python core.

Responsibilities:

- persist dynamic output schemas safely
- implement `IGH_VariableParameterComponent`
- expose stable `.gha` artifacts for Rhino / Grasshopper

Build output:

- `dist/grasshopper/YingZao.GH.gha`
